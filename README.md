# Telegram Post Link Bot

This bot captures posts from private groups (where it is admin),
generates a unique link for each post, and sends the post to users
via bot link with auto-delete after 5 minutes.

## Features
- Unlimited private groups
- No group ID required
- Unique link per post
- Auto delete after 5 minutes
- Railway compatible

## Deploy
- Python 3.10+
- Set ENV variables:
  API_ID, API_HASH, BOT_TOKEN
